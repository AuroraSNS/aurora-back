 <h3 align="center"> 오늘 당신의 날씨는 어떤가요? <br/><br/>
Aurora에서 당신의 기분을 날씨로 마음껏 표현해보세요. <br/><br/><br/> </h3>
 
-------------------------------------
<br/><br/>

#### 오늘따라 기분이 유독 좋아 어딘가에 이 기분을 기록해두고 싶은 적이 있나요?
#### 아니면 기분이 좋지 않아서 위로를 받고 싶은 적이 있나요?
#### Aurora에서는 여러분의 기분을 날씨로 담아 게시물을 올릴 수 있어요.
